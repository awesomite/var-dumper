## Why?

To set limit size of output of printed variable.

## Usage

```php
<?php

use Awesomite\VarDumper\LightVarDumper;
use Awesomite\VarDumper\Dumper;

$varDumper = new LightVarDumper();
$varDumper
    ->setMaxChildren(20)
    ->setMaxDepth(5)
    ->setMaxStringLength(200);

$varDumper->dump($GLOBALS);

// or easier way
Dumper::dump($GLOBALS);
```

## Installation
